
# SAFE Sketches

Todos los archivos modificables de este repositorio son con extensión `.excalidraw`

**Prototipo de Figma (solo puedes verlo si tienes acceso al figma)**: [SAFE_Sketches](https://www.figma.com/proto/NiwR71SCnzhLOaf42uKUry/SAFE_Sketches?node-id=5-3&p=f&t=CjPW5oTCBtEzZlSM-1&scaling=min-zoom&content-scaling=fixed&page-id=0%3A1&starting-point-node-id=5%3A3).

## Guía  para este repositorio

1. Entra a [Excalidraw](https://excalidraw.com/).

2. Haz clic en el ícono de menú, ubicado en la esquina superior izquierda.

3. Selecciona la opción **Abrir** o **Open**.

4. Busca el `.excalidraw` que desees abrir de este repositorio.

5. Haz clic en **Abrir**.

6. El archivo se cargará automáticamente en el lienzo de Excalidraw.

---

## Que hacer despues de terminar de editar el `.excalidraw`

### 1. Guardar el dibujo como archivo `.excalidraw`

1. Haz clic en el ícono de menú, ubicado en la esquina superior izquierda.

2. Selecciona la opción **Guardar en...** o **Save to...**.

3. Elige la opción **Archivo** o **File**.

4. Excalidraw descargará un archivo con extensión `.excalidraw`.

5. Guarda ese archivo con el nombre de la ruta separado por **"_"** y **"_safe"** al final.

---

### 2. Guardar el dibujo como PNG en la máxima resolución

1. Haz clic en el ícono de menú, ubicado en la esquina superior izquierda.

2. Selecciona la opción **Exportar imagen** o **Export image**.

3. En la ventana de exportación, selecciona el formato **PNG**.

4. En la opción de escala o tamaño, selecciona el valor más alto disponible, normalmente **3x**.

5. Dejale el fondo blanco sin dark mode.

6. Haz clic en **Download** o **Descargar**.

---

### 3. Subir a Github

1. Si modificaste un `.excalidraw` ya existente entonces borra su respectiva foto y `.excalidraw` y **sube las nuevas**.

2. Si es una ruta/pantalla nueva entonces solo **clasificala como publica o privada** y subela con el mismo formato de las demas.

3. Dale un **nombre explicativo al commit** para saber cual subiste o modificaste.
